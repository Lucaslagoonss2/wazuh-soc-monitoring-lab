# Wazuh SOC Monitoring Lab

A practical SOC monitoring home lab using Wazuh SIEM/XDR and Sysmon.

## Objectives

- Security Monitoring
- Alert Investigation
- Windows Event Analysis
- MITRE ATT&CK Mapping
- SOC Analyst Skills

## Scenarios

- PowerShell Obfuscation
- Brute Force Authentication Failures
- Scheduled Task Persistence
- SMB Authentication Failures
